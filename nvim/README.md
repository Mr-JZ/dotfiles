# NeoVim configuration

I have updated my editor configuration as of 02/03/2023.

## Installation

You have first to install [packer](https://github.com/wbthomason/packer.nvim) with the script.
After that you can start nvim, if the packages aren't synced you can type `:PackerSync`.

## Lessons

1. To make the insert mode in nvim more efficient, you can use the
`<leader> = " "` mapping to automatically insert spaces instead of
waiting for each space to be pressed. This can improve the editing
experience and make it feel less clunky.
